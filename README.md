# RGB Express

A WIP solver for the game RGB Express.

## Usage

Build:

```
mvn install
```

Run all levels:
```
java -jar target/rgbexpress-1.0-SNAPSHOT.jar 
```

Run a specific level:
```
java -jar target/rgbexpress-1.0-SNAPSHOT.jar levels/1/1/04
```
